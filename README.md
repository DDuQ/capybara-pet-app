# Capybara Pet App

Welcome to the Capybara Pet App! This application allows users to create and manage virtual capybara pets, interact with them, and earn achievements. Built with .NET and C#, this project aims to provide a fun and engaging experience for users.

## Features

- **User Management**: Register, log in, and manage user profiles.
- **Capybara Avatars**: Create, customize, and track stats of virtual capybara pets.
- **Interactions**: Feed, play with, and care for your capybara.
- **Virtual Items**: Manage and purchase items for your capybara.
- **Achievements and Rewards**: Earn achievements and receive rewards for milestones.

## Architecture

This application follows a clean architecture pattern to ensure separation of concerns and maintainability. The main layers are:

1. **API**: Handles HTTP requests and responses.
2. **Application**: Contains business logic and services.
3. **Domain**: Represents the core entities and business rules.
4. **Infrastructure**: Manages data access and external integrations.
